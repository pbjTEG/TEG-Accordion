TEG Accordion

Add accordion folding to a block of HTML generated by a content management system (CMS).

Provided

  • teg-accordion.js
  • teg-accordion.css (minimum styles to make the folding work)

Installation

Link required JavaScript and CSS.

   <link rel="stylesheet" href="../src/teg-accordion.css" type="text/css" />
   <script src="../src/teg-accordion.js" type="text/javascript"></script>

In a script element or .js file, initialize the TEGAccordion object. See options below. 

	window.myAccordion = new TEGAccordion({options});

Options

Defaults are for Ticket Holder fields on Engaging Network's Events module.

Option Type Default Description
categorySelector String '.en__registrants__ticket' CSS selector to identify a block of HTML containing elements to fold.
categoryHeadSelector String '.en__registrants__ticketHead' CSS selector to identify the heading of a block of HTML elements to fold. This element will act as the button.
groupSelector String '.en__registrants__group' CSS selector to identify the a block of HTML elements that are hidden when folded.
groupIsCatChild Boolean true Determines if the items identified by groupSelector are children of the items identified by categorySelector above. This alters the syntax used to find the folding content of each category.
useAria Boolean true If true then aria-expanded, aria-labelledby, aria-controls will be added to the appropriate HTML elements. If the CMS doesn't generate id attributes, these will be generated.
openClass String 'open' The CSS class name applied to the elements identified by groupSelector when a heading is clicked.
exclusive Boolean false If true then only one item will be open at a time. Any other open item will be automatically closed.
